## DOCX：开放共享科研记录行动倡议

我们积极响应[科研开源计划（DOCX）](https://mmcheng.net/docx/)。如今开源不仅仅是开放源码，还包括数据集、模型、教程和实验记录。我们也在探讨其它类别的开源方案和协议。

希望大家了解这个倡议，把这个倡议与自己的兴趣点结合，做点力所能及的事情。每个人的微小的贡献，汇聚在一起就是整个开源生态。


## 下载

### Docker

```
docker pull apachecn0/freelearn-ds-pt2-zh
docker run -tid -p <port>:80 apachecn0/freelearn-ds-pt2-zh
# 访问 http://localhost:{port} 查看文档
```

### PYPI

```
pip install freelearn-ds-pt2-zh
freelearn-ds-pt2-zh <port>
# 访问 http://localhost:{port} 查看文档
```

### NPM

```
npm install -g freelearn-ds-pt2-zh
freelearn-ds-pt2-zh <port>
# 访问 http://localhost:{port} 查看文档
```
